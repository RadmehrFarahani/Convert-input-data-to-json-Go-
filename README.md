# Convert-input-data-to-json-Go-
Student Info to JSON Converter

This is a small project written in the Go programming language.

The program collects student information from the user (name, major, and score) and saves it as a JSON file. The data is stored using a struct and converted to JSON before being written to a file.

-Concepts Used :

In this project, several Go concepts are practiced:

Functions and passing arguments

Structs and a constructor function (New)

Methods for displaying and saving data

Error handling using the error type

Working with multiple packages (main and info)

Standard Libraries Used

-strings for string processing :

errors for creating error messages

encoding/json for converting structs to JSON

os for writing files

bufio for reading user input

-Purpose of the Project :

The purpose of this project is to practice basic Go concepts, work with structs and JSON, handle user input, and organize a project into multiple packages while building a simple CLI application.
